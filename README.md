
# Тема проекта: игра "Угадай домашнее животное"  
(чтобы уменьшить количество возможных видов животных мы решили остановиться только на домашних)

Игра-угадайка с самообучающимся деревом решений. Программа задаёт вопросы, угадывает загаданное животное и запоминает новых зверей.

## Игровой процесс

1. **Загадайте животное** 
2. **Отвечайте на вопросы** 
3. **Когда бот выдаст предположение**:
   - Если угадал, нажмите **"Да, всё верно"**. Игра завершится победой.
   - Если не угадал, нажмите **"Нет, не угадали"**.
4. **Форма обучения** (появляется при неверном предположении):
   - Введите **название животного**, которое загадали.
   - Придумайте вопрос, отличающий ваше животное от того, что предположил бот.
   - Укажите правильный ответ ("Да" или "Нет") на этот вопрос для вашего животного.
   - Напишите интересный факт о нём (показывается при следующих угадываниях).
5. После сохранения бот запомнит новое животное и в следующий раз сможет его отгадать.

**Кнопка "Вернуться назад"** на экранах вопроса и предположения позволяет откатиться на один шаг.

## Архитектура

Проект построен вокруг **бинарного дерева решений**. Каждый узел – это вопрос, каждый лист – животное + факт.

| Файл | Назначение и детали |
|------|----------------------|
| **models.py** | Классы `Node` (вопрос + ссылки `yes`/`no`) и `Leaf` (название животного + факт). |
| **game_engine.py** | Содержит класс `GameEngine` с полями `current_node` (текущая позиция в дереве) и `history` (стек пройденных узлов для отката). Методы: `start()` – сброс, `answer()` – переход по ответу, `go_back()` – возврат, `learn_new_animal()` – замена неправильного листа на узел с вопросом и двумя листьями (старое + новое животное). |
| **persistence.py** | Функции `_to_dict()` / `_from_dict()` для рекурсивного преобразования дерева в JSON и обратно. `create_default_tree()` – начальное дерево из 15 животных. `save_game()` и `load_game()` сохраняют/загружают дерево в `savegame.json`. |
| **gui_ui.py** | Интерфейс на `tkinter`: главное меню, экран вопроса, экран предположения, форма обучения. Обрабатывает нажатия кнопок и вызывает методы движка. |
| **main.py** | Точка входа: загружает дерево, создаёт движок и GUI, запускает главный цикл. |
| **savegame.json** | Автоматически создаётся при первом сохранении. Хранит всё дерево (включая добавленных пользователем животных) и обновляется при закрытии программы. |

**Механика обучения:** когда бот ошибся, его текущий лист (неправильное животное) заменяется на новый узел с вопросом от пользователя. Оба листа – старое и новое – становятся его детьми. Благодаря этому дерево разрастается и покрывает всё больше животных.
